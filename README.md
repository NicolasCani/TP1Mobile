# TP1Mobile

Pour l'application SNCF, la clé API Navitia est configurée via le fichier local.properties. Par mesure de sécurité et conformément aux bonnes pratiques de développement, ce fichier n'est pas partagé sur le dépôt. Il sera nécessaire d'utiliser votre propre clé (renseignée sous la variable SNCF_API_KEY) pour bénéficier de l'intégralité des fonctionnalités de recherche d'itinéraires.
